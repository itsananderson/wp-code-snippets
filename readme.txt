Code Snippets allows you to define custom "keys" and replace them with predefined "values".

== Description ==

Code Snippets makes it easy to add expanding snippets to your code.
Simply create a snippet in the easy to use interface and then add the shortcode to your posts to make use of the snippet expansion.

== Installation ==

Upload the Code Snippets plugin to your blog, Activate it, then go to Settings -> Code Snippets and define your Snippets.
